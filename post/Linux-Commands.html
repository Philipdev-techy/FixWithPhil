10 Linux Commands Every Beginner Should Know

Posted on July 17, 2025

  1. pwd – Show current directory
  2. ls – List files and folders
  3. cd – Change directory
  4. mkdir – Create a folder
  5. touch – Create a new file
  6. rm – Delete files/folders
  7. mv – Move or rename files
  8. cp – Copy files
  9. man – Show manual/help
  10. whoami – Show current user

Practice these daily and you'll become more confident using Linux!
